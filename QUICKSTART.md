# SleepAI MVP - 快速开始

## 🚀 立即体验（无需配置）

### 1. 启动项目

```bash
npm run dev
```

### 2. 访问应用

打开浏览器访问：http://localhost:3000

### 3. 测试流程

1. **填写问卷** - 回答 5 个简单问题
2. **查看结果** - 系统会生成模拟的催眠脚本
3. **播放器** - 查看播放器界面（模拟音频）
4. **反馈** - 填写反馈问卷

## ⚙️ 完整功能（需要配置 API Key）

### 配置 OpenAI API Key

编辑 `.env` 文件：

```env
OPENAI_API_KEY="your_openai_api_key_here"
```

### 生成真实的催眠音频

配置 API Key 后，系统会：
1. 调用 GPT-4 生成真实的催眠脚本
2. 调用 TTS 生成真实的音频文件
3. 保存到 `public/audio/` 目录

## 🐛 常见问题

### Q: 启动失败
A: 确保已安装依赖：`npm install`

### Q: 无法生成音频
A: 检查 `public/audio/` 目录是否有写入权限

### Q: API 调用失败
A: 确保 `.env` 文件中配置了正确的 API Key
